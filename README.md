# Portal Predictor (MC 1.16.5)

Authors: Tuma, Tymamatyty

## EN
Pick point 1 and point 2 for the portal placement target.

- Calculates what blocks to clear.
- Calculates what blocks to place so portal appears at the desired location.

Status: still rough and needs polish, but works when used correctly.

## RU
Выбираешь точку 1 и точку 2 для желаемого появления портала.

- Мод считает, какие блоки нужно расчистить.
- Мод считает, какие блоки нужно поставить, чтобы портал появился ровно там, где нужно.

Статус: мод сырой и требует полировки, но в целом работает при правильном использовании.
